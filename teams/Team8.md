## Game next directions
So here will be the list of the new thing we want to add to the game:

- 1. Achievement extend
     Description: So previously we had achievement that we can access with the main menu.
     Functionalities:
    - Achievement popup during a game
    - Achievement reward such as new spaceship color



- 2. Game mode
     Description: Previously the game was only playable in two-player mode and one game mode.
     Functionalities:
    - Possibility to choose between one player or two-player mode
    - A whole new pvp gamemode with to player shooting at each other


- 3. Customization
     Description: The possibility to choose spaceship color
     Functionalities:
    - A new gacha system, player will be able to buy in the shop a ticket to roll a new color
    - Player will be able to change his spaceship color in the main menu



- 4. Pause button
     Description: Player will be able to pause the game.


- 5. Portal system
     Description: A whole new system, portal will spawn in the map and player will teleport randomly in the map. Especially good in pvp game mode.
